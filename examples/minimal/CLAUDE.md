# Example: Minimal Setup

This is a minimal CLAUDE.md example for small projects.

## Overview

A simple web application built with Node.js and Express.

## Tech Stack

- Node.js 20+
- Express 4.x
- SQLite

## Conventions

- Prefer async/await over callbacks
- Run `npm test` before committing
